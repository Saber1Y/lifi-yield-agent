# **Build autonomous agents that move capital across chains**

AI agents are becoming the new users of crypto.

They trade. They farm yield. They rebalance portfolios. They execute strategies faster and more consistently than humans ever could.

LI.FI gives agents the ability to move capital across chains, protocols, and ecosystems through a single unified interface.

This Vibeathon is about proving that future.

We are inviting builders to create autonomous agents powered by LI.FI.

---

# Objective

Build an AI agent that uses LI.FI to execute real on-chain strategies.

Your agent should independently make decisions and use LI.FI to move funds across chains to achieve its goal.

This is not about calling a swap endpoint once. This is about building agents that actively use LI.FI as part of their execution loop.

Examples include but are not limited to:

- Yield farming agents that hunt and rebalance into the best yields
- Trading agents that trade memes, narratives, or signals across chains
- Arbitrage agents that exploit cross chain inefficiencies
- Competitive agents that try to outperform each other
- Treasury management agents that optimize capital allocation
- Fully autonomous agents managing their own portfolios

Be creative. The more interesting and autonomous, the better.

---

# Required: Use LI.FI AI toolkit

Your agent must integrate LI.FI using one or more of the following:

- [LI.FI MCP Server](https://docs.li.fi/mcp-server/overview)
- [LI.FI Agent Skills](https://skills.sh/lifinance/lifi-agent-skills/li-fi-sdk) installed into Claude, Cursor, Codex, or other coding assistants
- [LI.FI OpenClaw skill plugin](https://clawhub.ai/rhlsthrm/lifi-crosschain)
- LI.FI API, SDK, or Composer in an agentic execution flow

Your submission should clearly demonstrate that LI.FI is part of the agent’s decision making and execution.
